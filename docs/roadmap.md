# whgot roadmap

## Current phase
- stable local developer setup
- passing test suite and green lint
- local web UI for upload -> identify -> price -> listing draft
- saved sessions under `~/.whgot/sessions/`
- basic eval scaffold and benchmark contract

## Next priorities
1. richer review workflow in web UI: filtering, sorting, grouping
2. benchmark corpus growth across books/media, toys/collectibles, electronics
3. better free/public pricing adapters and provenance detail
4. stronger multi-item image handling beyond prompt-only batch mode
5. improved export formats with optional richer seller columns

## Explicit non-goals for now
- direct marketplace posting
- paid API dependency as the default path
- mobile-native app
- collaborative swarm workflows
